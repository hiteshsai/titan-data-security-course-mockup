# Titan course mockup — Implement Microsoft Purview Data Security

A visual mockup of a proposed new Titan Tech Deal Ready course for Microsoft partner consultants.

**[View the mockup →](https://hiteshsai.github.io/titan-data-security-course-mockup/)**

## What this is

A static HTML mockup of a proposed new course on Titan (`skillupwithlevelup.com`), the Microsoft partner skilling academy. The course title is "Implement Microsoft Purview Data Security" and contains two numbered modules:

1. **Recommend and deploy sensitivity labels** — Microsoft Purview Information Protection
2. **Recommend and deploy data loss prevention policies** — Microsoft Purview DLP

The layout mirrors a real Titan course interior, including:
- The exact module-card grid pattern from the existing Data Security & Compliance Tech Deal Ready Titan journey (`/journeys/46`)
- Sequential module locking (`Modules and their sections must be completed in order`)
- Articulate Rise SCORM-style content sections
- Status pills (Start / Locked / Resume / 0%)
- Side rail Modules / Assessments / Instructors tabs
- Right-side Progress / To-do / Announcements / Status info cards

## Module previews

Each module preview shows:
- **Section 1 — design**: A live iframe embed of the [SMB Data Security Guided Assistant](https://cj-scott.github.io/GuidedLabelingAssistant_v2/) (community-built tool)
- **Section 2 — deploy**: A placeholder video tile where the recorded walkthrough of the [Microsoft Purview Best Practice Deployment Toolkit](https://github.com/amirjafarian/SMBBestPracticeTool) (community-built) will sit
- A section script outlining what the partner watches, anchored by "Maya, a clinic operations lead" as the recurring customer persona
