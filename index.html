Outline
What the partner watches (~3 min, screencast)
  1. Re-run .\Deploy-PurviewBestPractice.ps1 — observe the two DLP policies created: one for Exchange, one for SharePoint + OneDrive. Both block external sharing of Confidential\AllEmployees.
  2. Show why match condition uses the label GUID (not the display name) so rename does not break the policy.
  3. Walk the Change Management Playbook: simulation first, then Day 0 / Day 5 / Day 30 promote-from-simulation gate.

  5. Validate via Microsoft Purview portal: policies published, DLP simulation events flowing.
How to read this mockup. Top half shows the new course landing page exactly as it would appear inside Titan. The two preview sections below show what each module looks like when a partner clicks Start — including the live SMB Data Security Guided Assistant embedded as the design tool, and placeholder video tiles where the recorded walkthroughs will sit. Naming follows Microsoft Learn terminology (sensitivity labels, data loss prevention policies, Microsoft Purview). Tool names appear only inside section content, never in titles, so future tool renames do not break the course.

How this scales when more assets ship

Modules are sequentially numbered inside the course. Each one is an Articulate Rise SCORM package (same authoring pattern as the existing 4.5-hour Specialize Data Security Consultation course in journey 46, which grew from one module to five over time).

Insider Risk becomes a partner motion

Becomes Module 3 inside this course. Same Rise template. Tagged as requiring the Microsoft Purview Suite add-on inside the module content.

Endpoint DLP joins the deployment baseline

Becomes a new section inside Module 2, or Module 4 if it grows large. No new course needed.

eDiscovery walkthrough is requested

Becomes Module 5 inside this course, or a sibling course in the same Tech Deal Ready journey if the depth warrants.

An assessment becomes mandatory

Added as the final numbered Module (matches the Consultation course pattern: "Post-training Skills Assessment") and surfaces on the Assessments tab automatically.
